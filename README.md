# Easy Select

![](misc/example.png)

一个用于快速生成 CSS 选择器的Chrome插件。


## 环境安装
```
npm install
```

### 开发环境编译并热更新
```
npm run serve
```

### 生产环境打包
```
npm run build
```

### 分析包组件大小
```
npm run analyze
```
